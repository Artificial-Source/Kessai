# Quick Start

After installation, use this short flow to validate your setup.

## 1) Configure Basics

Open Settings and set:

- Currency
- Theme

## 2) Add Your First Subscription

From Subscriptions:

1. Click add subscription
2. Enter name, amount, billing cycle, and next payment date
3. Save and verify it appears in list/grid views

## 3) Check Dashboard and Calendar

- Dashboard should show totals and trend data
- Calendar should show upcoming payment entries

## 4) Test Backup/Restore

1. Export data to JSON
2. Re-import the same file
3. Confirm subscriptions and categories remain intact
